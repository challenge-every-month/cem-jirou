# 機能要件 (Functional Requirements)

## 1. 概要
「Challenge Every Month (CEM)」コミュニティにおいて、メンバーが毎月の目標（チャレンジ）を設定し、その進捗を管理・表明するためのSlack Appです。

## 2. ユーザーロール
### チャレンジャー (Challenger)
- Slackを利用しているコミュニティメンバー。
- 毎月の目標を設定し、進捗を報告する。

## 3. 主要機能
### 3.1 挑戦者登録
- `/cem_register`: Appを利用するための初期登録を行います。

### 3.2 目標設定
- `/cem_new`: その月の新しいプロジェクトとチャレンジ（目標）を登録します。
- プロジェクトは複数のチャレンジをまとめるカテゴリーのような扱いです。

### 3.3 目標の編集・削除
- `/cem_edit`: 登録済みのプロジェクトやチャレンジの内容、期限などを修正します。
- `/cem_delete`: 登録済みのプロジェクトやチャレンジを取り消します。

### 3.4 目標の表明 (Publish)
- `/cem_publish`: その月の目標をチャンネルに公開（宣言）します。
- 公開することで他のメンバーからのフィードバックを得やすくします。

### 3.5 進捗報告 (Progress)
- `/cem_progress`: 月末までの間に、現在の進捗状況を報告します（中間報告）。

### 3.6 振り返り (Review)
- `/cem_review`: 月末にその月の目標達成度を振り返り、コメントを添えて報告します。

## 4. リマインダー機能 (自動通知)
- **月初 (1日〜5日頃)**: 今月の目標設定と表明を促す通知。
- **月中 (15日頃)**: 進捗報告を促す通知。
- **月末 (25日〜30日頃)**: 振り返りを促す通知。
- **未完了タスクリマインド**: 期限が近い、または未完了のタスクがある場合に個別に通知。

## 5. 旧バージョン (cem-app) からの継承点
- コマンド体系（`/cem_...`）は基本的に完全に踏襲します。
- 「月ごとの目標管理」というコンセプトを維持します。
